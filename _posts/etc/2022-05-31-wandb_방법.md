### 4. 코드에 wandb 사용할 수 있도록 코드 삽입!

4-1) wandb import 먼저!
~~~
import wandb
~~~

4-2) main() 함수의 맨 앞부분에 init() 으로 초기화해준다! 혹, main()을 따로 하지 않고 바로 파일에 박아버릴 경우엔 아래 예시와 같이 그냥 써주면 된다!
~~~
wandb.init()
~~~

4-3) args 변수 선언부 밑에 아래와 같이 추가해준다.

~~~
wandb.config.update(args)
~~~

4-4) model 선언 다음에 다음 코드를 추가한다.

~~~
wandb.watch(model)
~~~
